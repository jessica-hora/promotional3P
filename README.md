# promotional3P
Projeto para criação de campanhas promocionais 3P com subsídio (rebate)

## Publicação

A interface está publicada no branch `gh-pages` como artefato privado do repositório.
Não existe uma URL pública ativa neste momento.

Se você quiser habilitar a publicação pública no futuro, crie o segredo de repositório `PAGES_TOKEN` com permissão `pages:write` para permitir a configuração automática do branch de origem.
